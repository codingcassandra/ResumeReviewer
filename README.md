Resume Reviewer

This project is a full-stack AI application that compares resumes against job descriptions and returns structured, applicant tracking system style feedback. It uses Groq’s Llama model via a FastAPI backend to analyze skill alignment, identify missing qualifications, and generate recruiter-style recommendations. The goal is to simulate how modern applicant tracking systems evaluate candidates while still providing readable improvement suggestions.

Challenges Solved
- Designing a reliable prompt structure to ensure consistent LLM JSON output.
- Handling imperfect AI responses and building fallback parsing logic.
- Normalizing and comparing skills across different naming conventions.
- Structuring AI-generated results for clean frontend rendering.
- Deploying a full-stack AI application with a cloud-hosted backend.

Key Features
- Upload and parse resume files (PDF and DOCX support)
- AI-powered resume analysis using Llama via Groq API
- Job description matching with extracted skill comparison logic
- ATS-style scoring system (match percentage)
- Structured feedback including:
	- Matched skills
	- Missing skills
	- Overall match score
	- Verdict (Strong / Moderate / Weak match)
- Personalized improvement recommendations generated by the LLM
- Full-stack integration (React frontend + FastAPI backend)
- Deployed backend for live API usage
